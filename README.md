# TaskFlow - 할 일 목록 애플리케이션

## 실행 방법

이 프로젝트는 백엔드(Node.js)와 프론트엔드(React)로 구성되어 있습니다.

### 1. 백엔드 실행
1. `backend` 폴더로 이동합니다.
2. 다음 명령어를 입력하여 서버를 실행합니다:
   ```bash
   node index.js
   ```
   서버는 `http://localhost:5000`에서 실행됩니다.

### 2. 프론트엔드 실행
1. `frontend` 폴더로 이동합니다.
2. (최초 실행 시) 패키지를 설치합니다:
   ```bash
   npm install
   ```
3. 다음 명령어를 입력하여 애플리케이션을 실행합니다:
   ```bash
   npm start
   ```
   애플리케이션은 브라우저에서 `http://localhost:3000`으로 열립니다.

## 주요 기능
- 할 일 추가, 완료 토글, 삭제
- 필터링 (전체, 진행 중, 완료됨)
- 데이터 영속성 (서버측 JSON 파일 저장)
